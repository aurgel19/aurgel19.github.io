Personal Projects

Scrap Fishing

A game about a lonely boy collecting scrap to build a robot friend. Created as a solo project.

  • Player controls a heavy-duty magnet as it descends into the scrapyard
  • Magnet has a depth and weight limit which can be increased by upgrading it with collected scrap
  • Different types of scrap can be found and collected at different depths
  • Collected scrap can be used to build pieces of your robot friend

Soup for the Spirits

A game where the player is a chef who makes soup for recently deceased spirits. Created as a solo project to practice creating dialogue with Yarn Spinner.

  • Conversations between the chef and spirits are displayed through the dialogue system
  • Recipe list shows which ingredients can be used to create different kinds of soup
  • Player cooks soup by clicking and dragging ingredients into a pot

Not Enough Space

A game created as a solo project where the player must manuever their rocket through a series of caverns to reach a landing pad.

  • Player controls rocket thrusters with spacebar and changes direction with A and D keys
  • Hazards such as cavern walls and meteors can destroy your ship and must be avoided
  • Levels are completed when the player successfully reaches the landing pad

High Tide

A platformer game where the player must escape from a flooding cave. Created as a solo project to practice implementing platformer movement mechanics.

  • Player traverses caverns by running, jumping, and climbing ladders
  • Spike hazards and slime enemies can kill the player
  • Water slowly fills the cave and can drown the player if they take too long

Breakout Clone

A Breakout clone made as a solo project using Unity. Added a variety of powerups:

  • Increase paddle size
  • Decrease paddle size
  • Spawn another ball
  • Increase paddle speed
  • Add 1000 bonus points

Student Projects

Pong 2.0

Created as a solo assignment for class. A clone of the classic Pong game made using Java as a solo assignment.

  • Local co-op where two players control each paddle
  • Ball speed increases with every hit, capping at twice its original speed
  • Added three powerups: increase ball size, decrease ball size, and spawn new ball

Plane Dodger

A Flappy Bird clone made using Java. Created as an independent assignment.

  • Player taps the up arrow to keep the plane in the air
  • Stars can be collected to earn points
  • Enemy planes fly from the opposite side of the screen and can crash into the player
  • Player's highest score is saved and displayed on the main menu

Space Rocks

A clone of the classic Asteroids game made using Java. Created independently for an assignment.

  • Player controls a spaceship and shoots lasers to destroy asteroids
  • Wraparound mechanics where the spaceship and asteroids move from one side of the screen and appear on the other
  • Spaceship can be teleported to a random location by pressing 'H'
  • Powerups can be used to create a shield around the spaceship
